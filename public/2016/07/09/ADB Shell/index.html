ADB Shell 学习

又是一个周末了,最怕闲来无事的周末，于是给自己找点事情做，ADB shell这一块一直没有系统的学习，正好趁此机会学习。

怎么使用ADB shell命令

首先找到Andorid的SDK安装路径或者单独安装ADB组件，sdk中adb路径在<Android SDK>platform-tools 文件夹下。

ADB Debugging命令

adb devices

列出连接的设备

adb devices [-l] ‘-l’

参数用于指定需要列出的设备

返回结果

adb forward

重定向连接,需要开启设备的USB debugging模式

adb forward <local> <remote>
adb froward –no-rebind <local> <remote>

作用同上，但是如果已经连接就会失败

adb forward –remove <local>

删除指定连接的设备

adb forward –remove-all

adb kill-server

终止adb服务进程 如果服务在运行则终止

无线连接命令

adb disconnect

断开通过TCP/IP连接的设备

adb disconnect [<host> [:<port>]]

不带参数则断开所有TCP/IP连接的设备

adb usb

App包的管理命令

adb install

安装Android应用到设备，需要指定需要安装的 .apk 文件的全路径

adb install [option] <path>

adb install-multiple

一次安装多个apk文件

adb install-multiple [-lrtsdpg] <file…>

参数用法和adb install相同

adb uninstall

从设备中卸载app

adb uninstall [-k] <package>

参数 -k 表示保留缓存和数据

adb shell pm list packages

打印出设备安装的所有包信息，可选参数用于过滤要显示的包名

adb shell pm list packages [options] <FILTER>

adb shell pm path

打印制定APK的路径

adb shell pm path <PACKAGE>

返回结果

adb shell pm clear

删除所有有关的数据

adb shell pm clear <PACKAGE>

返回结果
